# varconf-docker
>  分布式配置中心dockerfile

[![Build Status](https://travis-ci.org/varconf/varconf-docker.svg?branch=master)](https://travis-ci.org/varconf/varconf-docker)

## 使用说明
### 构建
```sh
docker build -t varconf
```

### 运行
```sh
docker run -p 8088:8088 varconf
```
